# Análisis de conversación digital — Pancho Pelayo

Reporte interactivo de análisis de comentarios de Facebook para la página **FranciscoPelayoCovarrubias** (11 publicaciones, 1,726 comentarios analizados, corte al 17 de julio de 2026).

El reporte incluye:

- Análisis de sentimiento por publicación y consolidado
- Conteo de reacciones, comentarios y compartidos
- Temas más repetidos en los comentarios
- Principales críticas y áreas de oportunidad
- Atributos positivos reconocidos por la audiencia
- Detección de ataques orquestados / comportamiento atípico
- Comparativo de narrativas (contraste, propuestas, cercanía, identidad sudcaliforniana, liderazgo)
- Nube de palabras y comentarios representativos por pieza
- Resumen ejecutivo y recomendaciones

## Ver el reporte

Abrir `index.html` en el navegador, o visitar la versión publicada en GitHub Pages.
